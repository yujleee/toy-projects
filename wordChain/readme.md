## 🖥 기술 스택

### Front-end

- `HTML5`, `SCSS`, `JavaScript`
- Vanilla JS로 기능 구현
- SCSS 부트스트랩으로 레이아웃 구성

<br/>
<br/>

## 🗂 디렉토리 구조

```
    📂css
    ┗ 📜main.css
    📂js
    ┗ 📜main.js
    📂scss
    ┣ 📂components
    ┗ 📜main.scss
    📜index.html
```

- css: 컴파일된 css 파일
- js: 기능 구현 js 파일 (리팩토링 예정)

<br/>
<br/>

## ✨ 구현

### 끝말잇기 시작
